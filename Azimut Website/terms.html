Description du service

Azimut est une application d'actualités qui affiche les événements mondiaux sur une carte interactive. L'Application permet de :

  • Consulter les actualités mondiales géolocalisées sur une carte
  • Filtrer les événements par continent, catégorie et langue
  • Suivre votre historique de lecture
  • Utiliser la fonction FastRecap pour parcourir rapidement les actualités
  • Rechercher des événements spécifiques
