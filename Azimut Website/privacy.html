Politique de Confidentialité

Dernière mise à jour : 3 février 2026

Introduction

Bienvenue sur Azimut ("nous", "notre", "l'Application"). Nous nous engageons à protéger votre vie privée et vos données personnelles. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application.

Données collectées

Données que nous collectons automatiquement

  • Données d'utilisation : Informations sur la façon dont vous utilisez l'application (articles consultés, filtres utilisés, temps passé dans l'application)
  • Données techniques : Type d'appareil, version iOS, identifiants anonymes pour les analyses
  • Données de localisation : Uniquement si vous l'autorisez, pour personnaliser les actualités de votre région

Données que vous fournissez

  • Préférences : Vos choix de filtres (continents, catégories, langues)
  • Historique de lecture : Articles marqués comme lus (stockés localement sur votre appareil)

Utilisation des données

Nous utilisons vos données pour :

  • Afficher les actualités sur la carte interactive
  • Mémoriser vos préférences et votre historique de lecture
  • Améliorer les performances et la stabilité de l'application
  • Gérer votre abonnement et supprimer les publicités pour les abonnés
  • Fournir les widgets pour les utilisateurs abonnés

Stockage des données

La majorité de vos données sont stockées localement sur votre appareil :

  • Historique des articles lus
  • Préférences de filtres
  • Cache des actualités pour utilisation hors ligne

Les données d'abonnement sont gérées de manière sécurisée par Apple via l'App Store.

Services tiers

Notre application utilise les services tiers suivants :

  • EventRegistry API : Pour récupérer les actualités mondiales
  • Apple MapKit : Pour afficher la carte interactive
  • Apple StoreKit : Pour gérer les abonnements in-app
  • Services de publicité : Pour les utilisateurs non-abonnés (désactivés avec un abonnement actif)

Publicités

L'application affiche des publicités pour les utilisateurs sans abonnement. Ces publicités peuvent utiliser des identifiants publicitaires pour personnaliser le contenu affiché. Les abonnés (Weekly ou Monthly) bénéficient d'une expérience sans publicité.

Vos droits

Conformément au RGPD et aux lois applicables, vous avez le droit de :

  • Accéder à vos données personnelles
  • Rectifier vos données inexactes
  • Demander la suppression de vos données
  • Retirer votre consentement à tout moment
  • Porter plainte auprès d'une autorité de contrôle (CNIL en France)

Pour supprimer vos données locales, vous pouvez simplement désinstaller l'application.

Sécurité

Nous prenons la sécurité de vos données au sérieux. Les communications avec nos serveurs sont chiffrées via HTTPS. Les données sensibles d'abonnement sont gérées exclusivement par Apple.

Enfants

Notre application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles d'enfants de moins de 13 ans.

Modifications

Nous pouvons mettre à jour cette politique de confidentialité de temps en temps. Nous vous informerons de tout changement significatif via l'application ou par d'autres moyens appropriés.

Contact
